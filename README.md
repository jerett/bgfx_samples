# bgfx sample
===

## compile 
---
```
git clone https://github.com/jerett/bgfx_samples.git
cd bgfx_samples
git submodule update  --init  --recursive
mkdir build
cd build
cmake ..
```

## note
---
The samples support Windows and Mac at present, and I plan to add iOS and Andorid support later.
